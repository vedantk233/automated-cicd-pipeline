About This Project

This project demonstrates a complete Continuous Integration and Continuous Deployment (CI/CD) pipeline using GitHub Actions and AWS EC2. Whenever new code is pushed to GitHub, GitHub Actions automatically connects to the AWS EC2 server using SSH and deploys the latest version of the website.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Git
  • GitHub
  • GitHub Actions
  • AWS EC2
  • Ubuntu
  • Apache2

Project Workflow

Developer → GitHub → GitHub Actions → AWS EC2 → Apache Server → Live Website
